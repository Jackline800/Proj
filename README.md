# 🛰️ insoflow (向日葵衛星光流法日射量預報系統維護面板)

*透過直觀、單一網頁（Single-page）的互動式架構，完整呈現向日葵衛星光流法日射量預報之檔案路徑、8大作業流程鏈結與 Crontab 自動清理排程指令。*

### 📊 光流法日射量預報作業流程與路徑對照系統 (insoflow)
整合全域環境變數、各作業階段之輸入/輸出目錄、IDL與Python核心程式放置位置，並支援關鍵字即時搜尋與類別篩選，提供運維工程師現場快速定位目錄，內建一鍵複製 crontab 每 10 分鐘主流程排程及自動維護清理指令。[cite: 1] <br>

## 🛠️ 架構說明

- **insoflow/**
  - **`index.html`**：核心前端監測面板。整合了全域環境變數對照表（`set_enviorn_variables_share.sh`）、程式檔放置表、Domain定義資料、以及 8 大資料處理流向（從 HSD 下載、轉檔、太陽幾何修正、光流場外推、範圍裁切至日射量反演模式）。[cite: 1]

- **insoinve/**
  - **`index.html`**：技術白皮書簡報網頁。詳載日射量反演系統之系統性優化軌跡、物理模組擴充歷程與動態品質防護機制。[cite: 2]
